Enter key value pairs of locations of JKDs to build tracks with. Possible keys are:

  • JDK1.3.1_HOME,
  • JDK1.4.2_HOME,
  • JDK1.5.0_HOME,
  • JDK1.6.0_HOME and
  • JDK1.8.0_HOME.

A key value pair should be separated from other entries by semicolon:

JDK1.4.2_HOME=/opt/jdk1.4.2;JDK1.6.0_HOME=/opt/jdk1.6.0_35
